Тестовое задание

Краткое описание
	Есть проект в Юнити 2017.1.0f3 в котором на сцене лежит плейн и простой макет ui. Нужно на примитивах сделать следующий функционал

Сделать UI из макета с 2d изображениями юнитов для спавна (A - типы юнитов)
Через drag&drop из UI сделать спавн юнитов на плейне (через 3d примитивы заданного цвета) 
Пока идет перетаскивание объекта копия спрайта движется за мышкой (без отставаний от мышки)
По окончании перетаскивания юнит спавнится на плейне в позиции под мышкой, если окончание перетаскивания происходит за пределами плейна, или над элементами UI - drag&drop должен отменяться, а спавна не проиходит
После спавна юниты нужно двигать с рандомной скоростью вперед от камеры, по окончанию плейна юнити нужно уничтожать 
К юнитам нужно привязать TextField в UI  в тексте которого записан текущий номер юнита (сколько юнитов уже было создано вместе с ним) 
TextField нужно двигать за юнитами (текстфилды должны двигаться плавно без отставаний)
Если что-то можно оптимизировать - нужно оптимизировать
